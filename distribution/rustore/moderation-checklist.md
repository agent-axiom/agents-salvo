- https://www.rustore.ru/help/developers/publishing-and-verifying-apps/app-publication
- https://www.rustore.ru/help/developers/publishing-and-verifying-apps/requirement-apps
- https://www.rustore.ru/help/guides/sign-apk

## До сборки

- [ ] Все тесты и coverage проходят на релизном commit.
- [ ] `versionName` равен `1.0.0`, `versionCode` равен `1`.
- [ ] Package name равен `io.github.agentaxiom.salvo`.
- [ ] Постоянный release keystore сохранён в защищённом месте и имеет отдельную резервную копию.
- [ ] SHA-256 fingerprint сертификата совпадает с `EC:49:72:02:0B:0B:43:7F:83:BD:29:31:5C:52:60:E2:C7:5C:83:4E:D5:C4:E3:65:01:21:CD:87:8C:D7:14:36` и записан рядом с резервной копией.
- [ ] GitHub Environment `rustore-production` разрешает деплой только с `main`, а его Secrets `RUSTORE_KEYSTORE_BASE64`, `RUSTORE_STORE_PASSWORD`, `RUSTORE_KEY_ALIAS`, `RUSTORE_KEY_PASSWORD` настроены.

## Owner action: аккаунт и правовые поля

- [ ] Владелец зарегистрирован и верифицирован в RuStore Консоли через собственный VK ID.
- [ ] Владелец указал реальные имя, email и иные обязательные контактные данные.
- [ ] Владелец подтвердил права на название, код, изображения, звуки и другой контент приложения.
- [ ] Владелец проверил применимые требования и подтвердил статус оператора персональных данных.
- [ ] Владелец настроил получение согласия на обработку персональных данных в форме, соответствующей применимым требованиям.
- [ ] Возрастная категория 6+ и декларации игровых материалов подтверждены владельцем.
- [ ] Приложение отмечено как бесплатное, без рекламы, покупок и подписок.

Не отправлять приложение на модерацию, пока юридические пункты выше не подтверждены владельцем. Не записывать персональные реквизиты в этот репозиторий.

## Артефакты

- [ ] Ручной workflow `Build RuStore Release` завершился успешно.
- [ ] Скачаны `app-release.apk`, `app-release.apk.sha256`, `app-release.aab`, `app-release.aab.sha256`.
- [ ] SHA-256 APK совпадает с checksum-файлом.
- [ ] `apksigner verify --verbose --print-certs app-release.apk` завершился успешно.
- [ ] Fingerprint APK совпадает с fingerprint постоянного release key.
- [ ] APK установлен на чистое устройство командой `adb install app-release.apk`.
- [ ] В итоговом APK присутствуют только `INTERNET`, `ACCESS_NETWORK_STATE`, `VIBRATE` и внутреннее signature-разрешение приложения.

Для первой публикации использовать APK. AAB сохраняется как проверенный дополнительный артефакт; для его загрузки RuStore требует отдельно добавить подпись приложения.

## Приёмка на устройстве

- [ ] Приложение запускается без сети; доступны бой против агента и режим на одном устройстве.
- [ ] Поля 8×8 и 10×10 помещаются по ширине экрана в основном боевом сценарии.
- [ ] Поле 16×16 использует управляемую внутреннюю панораму или масштаб, не создавая горизонтальную прокрутку всей страницы.
- [ ] Ручная и случайная расстановка работают, корабли не соприкасаются.
- [ ] Попадания, промахи и потопленные корабли визуально различимы в светлой и тёмной темах.
- [ ] Музыка, эффекты, отключение звука и виброотклик работают.
- [ ] Telegram-вход открывается, возвращает пользователя в приложение и сохраняет сессию после перезапуска.
- [ ] Профиль, лидерборд и личный архив загружаются после входа.
- [ ] Две собственные Telegram-учётные записи могут создать комнату, сыграть онлайн-бой и открыть повтор.
- [ ] Выход из аккаунта удаляет локальную сессию.
- [ ] Ссылка на уведомление о конфиденциальности открывается из настроек.
